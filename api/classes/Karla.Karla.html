raw()

raw(string  $program, string  $arguments = "") : string

Run a raw ImageMagick command

Karla was never intended to wrap all of the functionality of ImageMagick and likely never will, you will from time to time need to write arguments to ImageMagick like you would have done directly in the consol.

Parameters

string $program

Imagemagick tool to use
